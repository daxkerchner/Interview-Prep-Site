Technical interview prep class

Learn the skills to pass an interview!

This course will cover data structures and algorithms knowledge required to pass the interviews for most software jobs. The pacing involves just a few hours a day, designed for people who are job searching at the same time.

Schedule
(Monday-Friday, two weeks total):

  • 10:30am-11:30am ET: New material
  • 11:30am-1:30pm ET: Working through LeetCode or similar problems with live help from me

This course is for people who know how to code, but need the foundation to pass technical interviews given by most tech companies.

Concepts covered:

  • Time complexity
  • Using pointers in lists & strings
  • Algorithms: Sorting algorithms
  • Algorithms: Binary search
  • Data structures: Linked lists
  • Data structures: Trees
  • Data structures: Graphs
  • Data structures: Graphs on a grid
  • Recursion

This course will be taught in Javascript, but the concepts transfer easily to other languages. If you are most familiar with another language, contact me to confirm whether the course will be a good fit.

This course is for you if:

  • You know how to code but have not taken a Data Structures & Algorithms class
  • You're not sure where to start with LeetCode problems
  • You're not already familiar with most of the above concepts

This course is not for you if:

  • You already have a strong grasp of data structures and algorithms / you find LeetCode problems easy
    — if that describes you, you should book mock interviews to practice problem-solving live in an interview instead.
  • You are not yet familiar with a programming language
    — in that situation, you need to pursue the foundation first. My course assumes students have taken the equivalent of programming classes or a coding bootcamp
